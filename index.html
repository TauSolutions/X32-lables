X32 Compact Label Maker

Help -- This dialog...
Print -- Brings up the system print dialog
Load File -- Load an X32 Label file in JSON format
Save File -- Saves an X32 Label file in JSON format
Create Group -- Create a Label Group
Delete Group -- Delete a Label Group
SelectedName -- The name of the selected Group
Group Size -- The Outer cut size in mm of the button group
Buttons - Rows -- Number of rows/pairs of buttons
Buttons - Size -- The actual size of the buttons in mm
Buttons - Spacing -- Horizontal and Vertical Spacing in mm on Center of the Buttons
Buttons - Border -- The size of the button cutout border in mm
Buttons - Border Color -- Button Border color. Can be any valid CSS color. Color name (ie red, white, black, blue, etc) or HEX (ie #ABCDEF, #000, etc) or RGB (ie rgb(255, 128, 64), rgb(0,0,0), etc)

Usage:
Click Create to create a new Button Group.
Click the Button Group Title to change the text.
Click on any of the buttons to change their text.
Click on any Group to select it.
Group must be selected before it can be deleted.
This app auto saves the work into the browser cache and will load the next time the site is loaded.
Print will format remove the toolbox and format the buttons for proper printing.
